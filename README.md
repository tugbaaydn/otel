# Aydın Hotel Rezervasyon Yönetim Sistemi

Aydın Hotel Rezervasyon Yönetim Sistemi; otel rezervasyonlarını, müşteri bilgilerini, oda durumlarını, giriş-çıkış işlemlerini ve gelir takibini kolayca yönetmek için geliştirilmiş bir web tabanlı yönetim panelidir.

## Proje Hakkında

Bu proje PHP, MySQL, HTML, CSS ve Bootstrap kullanılarak geliştirilmiştir. Sistem otel personelinin rezervasyonları takip etmesini, odaların durumunu görmesini, müşteri kayıtlarını yönetmesini ve günlük işlemleri kolaylaştırmasını amaçlar.

## Kullanılan Teknolojiler

- PHP
- MySQL
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Bootstrap Icons
- Laragon
- HeidiSQL

## Özellikler

- Yönetici giriş sistemi
- Dashboard ekranı
- Toplam rezervasyon takibi
- Boş, dolu, temizlikte ve bakımda olan oda takibi
- Günlük gelir görüntüleme
- Rezervasyon ekleme, düzenleme ve silme
- Müşteri ekleme, düzenleme ve silme
- Oda ekleme, düzenleme ve silme
- Oda görseli ve oda özellikleri görüntüleme
- Katlara göre oda listeleme
- Oda filtreleme sistemi
- Giriş ve çıkış işlemleri
- Bildirim sistemi
- Raporlama ekranı
- Mobil uyumlu tasarım
